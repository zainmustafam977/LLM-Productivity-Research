02

International Research Standards

Formal hypothesis testing with standardized effect size measurement and statistical significance thresholds.

H₁: Speed Enhancement

LLM-powered autocomplete significantly accelerates task completion speed relative to a baseline control condition.

Autocomplete users implemented substantially more requirements within the fixed time window, demonstrating that inline AI code suggestions meaningfully enhance development velocity.

Effect Size Cohen's d = 1.25 (Large)
Significance p < 0.01 ✓
H₂: Quality Maintenance

AI-assisted code generation preserves comparable levels of maintainability to manually written code.

Maintainability Index scores across all three conditions show negligible variation, confirming that AI tools augment speed without compromising code structure or readability.

Effect Size Cohen's d = 0.12 (Negligible)
Significance p > 0.05 ✗
